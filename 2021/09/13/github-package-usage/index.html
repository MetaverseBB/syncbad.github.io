在Github Package上发布你的包

  1. 右上角头像 → setting → Developer settings → Personal access tokens
  2. Generate new toke
  3. 以上两步可以直接点击传送门到达 https://github.com/settings/tokens/new
  4. 选择期限
  5. 选中write:packages，系统会自动勾选其他必须权限
  6. 记录下来生成的Token
